今後のために1点コメントします。
役割や用途が一目瞭然になるような、変数名やメソッド名を使いましょう。
また、アクセス範囲の修飾子は最小限にするのがポイントです。

例えば、以下メソッドの定義は、

```
public static int tax(int price4, double price5)
```

以下のようにするとベターです。

```
private static int calculateTax(int price, double taxRate)
```

* このクラス内に限って使うメソッドのため、アクセス修飾子をprivateにしました。

* 税金を計算するメソッドなので、calculateTaxというメソッド名にしました。

* 「金額」「税率」を表現するため、price, taxRateという引数名にしました。
